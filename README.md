### Project Outline

- User Interface (HTML, CSS)

- Functionality
  - Adding Todo
  - Deleting Todo
  - Checking a Task
  - Total number of tasks

- Data
  - Tasks (Object storing tasks)
  - Task Details (ID, task, done etc.)

- Functions
  - addTodo
  - deleteTodo
  - checkTodo
  - showNotification
  - renderList
